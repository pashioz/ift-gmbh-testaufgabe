# IfT GmbH Testaufgabe

## Getting Started

Clone the repository.

    git clone https://github.com/pashioz/ift-gmbh-testaufgabe.git

Move into the project directory.

    cd ift-gmbh-testaufgabe

Install the dependencies

    npm install
    
    # or
    
    yarn

Start the application.

    npm run dev
    
    # or
    
    yarn dev

Enjoy!
